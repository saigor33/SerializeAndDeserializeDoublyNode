Проект был выполнен в Visual Studio 2010.
Инструкция по работе с приложением "createSerialize_DoublyNode":

1. Запуск программы.
2. Нажать на контрол "Добавить 5 элементов и сериализовать"
	Осуществится создание и добавление пяти элементов в двухсвязный список, а также произойдёт заполнения поля "Random" случайным эл-том текущего списка.
	В поле "Связи полученного списка" произойдёт отображение текущих значений списка в формате Item{<Значение Data>, <Значение Data след эл-та>, <Значение Data предыдущего эл-та>, <Значение Data случайного эл-та>.
	В поле "Сериализованный список" будет отображён текст(результат сереализации), полученный из созданного документа.
______________________________________________

3. Далее происходит процесс десериализаии. Для этого необходимо нажать на контрол "Десериализовать файл". Произойдёт считывания потока байтов из файла и отображение этих данных в поле "Содержимое файла".Затем произойдёт десериализация данных, в поле  "Связи после десериализации" осуществится вывод проверки связей списка путём применения того же метода, который был применён для проверки связей до сериализации.
